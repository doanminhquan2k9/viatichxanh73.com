🔧 Hướng Dẫn Cài Đặt Tool Trên Windows Xem Thêm ⏷

Bước 1: Tải Python

Truy cập trang web chính thức của Python để tải phiên bản mới nhất:
Tải Python 3.12
Sau khi tải về, mở file cài đặt để tiến hành cài đặt.

Bước 2: Cài đặt Python và kiểm tra

1. Mở file cài đặt Python đã tải về.

2. **Lưu ý quan trọng:** Tích chọn "Add Python to PATH" để tránh lỗi về sau.

3. Nhấn Install Now và chờ quá trình cài đặt hoàn tất.

4. Sau khi cài đặt xong, mở Command Prompt (cmd) và kiểm tra bằng lệnh sau:

python --version
pip --version

Nếu hiển thị phiên bản Python và Pip, nghĩa là bạn đã cài đặt thành công.

Bước 3: Cài đặt thư viện cần thiết

Mở Command Prompt (cmd) và nhập lệnh sau để cài đặt các thư viện cần thiết:

pip install requests pystyle bs4

Hệ thống sẽ tự động tải và cài đặt các thư viện này. Chờ quá trình hoàn tất.

Bước 4: Chạy Tool

Di chuyển đến thư mục chứa file tool (ví dụ: haidang.py) bằng lệnh:

cd đường_dẫn_đến_folder

Tiếp theo, chạy tool bằng lệnh:

python tên_file.py

Nếu gặp lỗi, thử kiểm tra lại các bước hoặc cập nhật pip bằng lệnh:

python -m pip install --upgrade Pip

Chúc bạn cài đặt thành công! 🚀
